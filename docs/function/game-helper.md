## Game Helper

`src/common/utils/GameHelper.ts`

### Set Pcode

```ts
setPcode(value);
```

Params:
value: `number`

### Get Pcode

```ts
getPcode();
```

### Get Base Pcode

```ts
getBasePcode();
```

return `string`

### Get Pcode Variant

```ts
getVariant();
```

example:

pcode = `m22`
return = `''`

pcode = `m22b`
return = `'b'`

pcode = `m23c`
return = `'c'`
