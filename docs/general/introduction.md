# Guide to `idnlive-game-modules`

## Tech Stack

Here's a curated list of packages that you should be at least familiar with before starting your project. However, the best way to see a complete list of the dependencies is to check [package.json](https://gitdewa99.com/idnlive/idnlive-game-modules/-/blob/main/package.json).

### Core

-   [ ] [React](https://react.dev/)
-   [ ] [Redux](http://redux.js.org/)
-   [ ] [Redux Toolkit](https://redux-toolkit.js.org/)
-   [ ] [i18next](https://www.i18next.com)
-   [ ] [React i18next](https://react.i18next.com)
-   [ ] [CSS Module](https://github.com/css-modules/css-modules)

### API Mocking

-   [ ] [Mirage](https://miragejs.com)

### Linting

-   [ ] [ESLint](http://eslint.org/)
-   [ ] [Prettier](https://prettier.io/)

## Project Structure

### `common/`

This folder contains components and modules that are present in all games.

### `game/`

This folder contains components and modules specific to one game.

### `api/`

This folder contains dummy API configurations for mocking the actual APIs on the server.
