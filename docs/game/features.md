## Features

Edit file `src/common/utils/Features.ts`

### Set Layout Game

```ts
LAYOUT_VERSION: 1 | 2 | 3,
```

### Enable Button Pattern in Landscape

```ts
BUTTON_PATTERN: boolean;
```

### Enable Modal Mini How To Play

```ts
MINI_HOW_TO_PLAY: boolen;
```

### Enable Animation Shuffle The Card

```ts
SHUFFLE_THE_CARDS: boolen;
```

## Enable Letter Box

```ts
LETTER_BOX: boolen;
```

## Enable Streaming Letter Box in Landscape Mode

```ts
STREAMING_LANDSCAPE_LETTER_BOX: true,
```

## Enable Animation Place Chip

```ts
CHIP_ANIMATION: true,
```
